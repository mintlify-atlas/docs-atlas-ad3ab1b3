The `Chassis` class provides a complete chassis control system with odometry tracking, PID control, and advanced motion algorithms for autonomous movement.

## Constructor

### Chassis (with horizontal distance sensor)

```cpp
Chassis(DriveTrain *dt, ChassConstants constants, pros::Imu *imu,
        pros::Rotation *vertEnc, pros::Rotation *horizEnc, 
        pros::Distance *horizDist)
```

Creates a new Chassis object with all sensors including a horizontal distance sensor.

<ParamField path="dt" type="DriveTrain*" required>
  Pointer to the DriveTrain object controlling the motors
</ParamField>

<ParamField path="constants" type="ChassConstants" required>
  Constants defining the physical dimensions of the chassis (see ChassConstants struct)
</ParamField>

<ParamField path="imu" type="pros::Imu*" required>
  Pointer to the IMU sensor for heading tracking
</ParamField>

<ParamField path="vertEnc" type="pros::Rotation*" required>
  Pointer to the vertical tracking wheel encoder
</ParamField>

<ParamField path="horizEnc" type="pros::Rotation*" required>
  Pointer to the horizontal tracking wheel encoder
</ParamField>

<ParamField path="horizDist" type="pros::Distance*" required>
  Pointer to the horizontal distance sensor for additional tracking
</ParamField>

## ChassConstants Struct

```cpp
struct ChassConstants {
    double horizWidth;
    double vertWidth;
    double trackDia;
    double vertDia;
    double horizDia;
};
```

<ParamField path="horizWidth" type="double">
  Horizontal distance from the tracking center to the horizontal tracking wheel (inches)
</ParamField>

<ParamField path="vertWidth" type="double">
  Horizontal distance from the tracking center to the vertical tracking wheel (inches)
</ParamField>

<ParamField path="trackDia" type="double">
  Diameter of the drive wheels (inches)
</ParamField>

<ParamField path="vertDia" type="double">
  Diameter of the vertical tracking wheel (inches)
</ParamField>

<ParamField path="horizDia" type="double">
  Diameter of the horizontal tracking wheel (inches)
</ParamField>

## MotionParams Struct

```cpp
struct MotionParams {
    bool async;
    int timeout = 3000;
    double vStart = 0;
    double vMin = 13;
    double vMax = 127;
    double angMin = 0;
    double angMax = 127;
    double settleRange;
    double settleTime;
    Exit* exit = nullptr;
    double rotationCut;
    double drift = 0;
    bool reverse = false;
    double within = 0;
    double slew = 0;
    double turnBias = -1;
    bool debug = false;
};
```

<ParamField path="async" type="bool">
  Whether the motion should run asynchronously (true) or block until complete (false)
</ParamField>

<ParamField path="timeout" type="int" default="3000">
  Maximum time in milliseconds to wait for motion to complete
</ParamField>

<ParamField path="vStart" type="double" default="0">
  Starting velocity for the motion
</ParamField>

<ParamField path="vMin" type="double" default="13">
  Minimum linear velocity during motion
</ParamField>

<ParamField path="vMax" type="double" default="127">
  Maximum linear velocity during motion
</ParamField>

<ParamField path="angMin" type="double" default="0">
  Minimum angular velocity during motion
</ParamField>

<ParamField path="angMax" type="double" default="127">
  Maximum angular velocity during motion
</ParamField>

<ParamField path="settleRange" type="double">
  Distance range within which the robot is considered settled
</ParamField>

<ParamField path="settleTime" type="double">
  Time in milliseconds the robot must remain within settleRange to be considered settled
</ParamField>

<ParamField path="exit" type="Exit*" default="nullptr">
  Custom exit condition for the motion (see Exit classes)
</ParamField>

<ParamField path="rotationCut" type="double">
  Distance at which to stop rotating towards the target
</ParamField>

<ParamField path="drift" type="double" default="0">
  Drift compensation factor for curved movements
</ParamField>

<ParamField path="reverse" type="bool" default="false">
  Whether to drive in reverse
</ParamField>

<ParamField path="within" type="double" default="0">
  Distance to stop before reaching the target
</ParamField>

<ParamField path="slew" type="double" default="0">
  Slew rate for velocity ramping
</ParamField>

<ParamField path="turnBias" type="double" default="-1">
  Bias for turn direction (-1 for automatic)
</ParamField>

<ParamField path="debug" type="bool" default="false">
  Enable debug output for motion
</ParamField>

## State Query Methods

### getPose

```cpp
Pose getPose()
```

Returns the current pose (position and heading) of the robot.

**Returns:** Current Pose with x, y position and heading

### getTheta

```cpp
double getTheta()
```

Returns the current heading of the robot in degrees.

**Returns:** Heading in degrees (0-360)

### isSettled

```cpp
bool isSettled()
```

Checks if the robot has settled at its target position.

**Returns:** True if settled, false if still moving

### waitUntilSettled

```cpp
void waitUntilSettled()
```

Blocks execution until the robot has settled at its target position.

### measureOffsets

```cpp
std::pair<double, double> measureOffsets(int iterations)
```

Measures and returns the encoder offsets by sampling sensor data.

<ParamField path="iterations" type="int" required>
  Number of samples to take for measurement
</ParamField>

**Returns:** Pair of vertical and horizontal encoder offsets

## Motion Methods

### turn

```cpp
void turn(double angle, MotionParams params)
```

Turns the robot to an absolute heading.

<ParamField path="angle" type="double" required>
  Target heading in degrees (0-360)
</ParamField>

<ParamField path="params" type="MotionParams" required>
  Motion parameters controlling the turn behavior
</ParamField>

**Example:**
```cpp
MotionParams params;
params.async = false;
params.timeout = 2000;
params.vMax = 100;
chassis.turn(90, params); // Turn to face 90 degrees
```

### turnTo

```cpp
double turnTo(Pt target, MotionParams params)
```

Turns the robot to face a target point.

<ParamField path="target" type="Pt" required>
  Target point to face (x, y coordinates)
</ParamField>

<ParamField path="params" type="MotionParams" required>
  Motion parameters controlling the turn behavior
</ParamField>

**Returns:** Final heading after turning

### driveAngle

```cpp
void driveAngle(double dist, double angle, MotionParams params,
                bool absolute = false)
```

Drives a specified distance at a given angle.

<ParamField path="dist" type="double" required>

</ParamField>

<ParamField path="angle" type="double" required>

## Usage Example

```cpp
// Initialize drive motors
pros::MotorGroup leftMotors({-1, -2, -3});
pros::MotorGroup rightMotors({4, 5, 6});

// Create drivetrain
keejLib::DriveTrainConstants dtConsts = {
    .gearRatio = 0.75,
    .cartridge = 600,
    .wheelDia = 3.25
};
keejLib::DriveTrain dt(&leftMotors, &rightMotors, dtConsts);

// Create sensors
pros::Imu imu(10);
pros::Rotation vertEnc(11);
pros::Rotation horizEnc(12);

// Create chassis constants
keejLib::ChassConstants chassConsts = {
    .horizWidth = 6.5,
    .vertWidth = 2.75,
    .trackDia = 3.25,
    .vertDia = 2.75,
    .horizDia = 2.75
};

// Create chassis
keejLib::Chassis chassis(&dt, chassConsts, &imu, &vertEnc, &horizEnc);

// Configure PID constants
keejLib::PIDConstants linConsts = {.kp = 8, .ki = 0, .kd = 0, .kf = 0};
keejLib::PIDConstants angConsts = {.kp = 5, .ki = 0, .kd = 0, .kf = 0};
chassis.setConstants(linConsts, angConsts);

// Start tracking
chassis.startTracking();

// Set starting pose
chassis.setPose({{0, 0}, keejLib::Angle(0, keejLib::HEADING)});

// Move to a point
keejLib::MotionParams params;
params.async = false;
params.vMax = 100;
chassis.mtpoint({24, 24}, params);

// Turn to face a direction
chassis.turn(90, params);
```
