## Overview

Differential drive control uses two separate PROS `MotorGroup` objects - one for the left side and one for the right side of your drivetrain. This allows independent control of each side, which is essential for turning and precise movement.

<Note>
keejLib's `DriveTrain` class uses two separate `pros::MotorGroup` objects to provide differential control. There is no special "differential group" wrapper class.
</Note>

## Setup

Create two separate motor groups - one for each side of the drivetrain:

```cpp
// Left side motors (typically reversed)
pros::MotorGroup leftMotors({-1, -2, -3});

// Right side motors (normal direction)
pros::MotorGroup rightMotors({4, 5, 6});
```

<Note>
Use negative port numbers to reverse motor direction. Typically, one side of your drivetrain needs to be reversed so both sides spin "forward" together.
</Note>

## Using with DriveTrain

The primary way to use differential drive with keejLib is through the `DriveTrain` class:

```cpp
#include "keejLib/chassis.h"

// Create motor groups
pros::MotorGroup leftMotors({-1, -2, -3});
pros::MotorGroup rightMotors({4, 5, 6});

// Define drivetrain constants
keejLib::DriveTrainConstants dtConsts = {
    .gearRatio = 0.6,      // Output/input gear ratio
    .cartridge = 600,      // Motor cartridge RPM
    .wheelDia = 3.25       // Wheel diameter in inches
};

// Create DriveTrain
keejLib::DriveTrain driveTrain(&leftMotors, &rightMotors, dtConsts);
```

## DriveTrain Methods

### spinVolts()

Apply voltages to left and right sides independently:

```cpp
// Using ChassVelocities struct
keejLib::ChassVelocities velocities = {100, 100};
driveTrain.spinVolts(velocities);
```

### spinAll()

Spin both sides at the same voltage:

```cpp
driveTrain.spinAll(127);  // Full speed forward
```

### spinLeft() / spinRight()

Control sides independently:

```cpp
driveTrain.spinLeft(127);   // Left side forward
driveTrain.spinRight(-127); // Right side backward
```

## Integration with Controller

The keejLib Controller's `drive()` method returns `ChassVelocities` that can be used with `DriveTrain`:

```cpp
pros::Controller prosController(pros::E_CONTROLLER_MASTER);
keejLib::Controller controller(prosController);

pros::MotorGroup leftMotors({-1, -2, -3});
pros::MotorGroup rightMotors({4, 5, 6});

keejLib::DriveTrainConstants dtConsts = {0.6, 600, 3.25};
keejLib::DriveTrain driveTrain(&leftMotors, &rightMotors, dtConsts);

void opcontrol() {
    while (true) {
        // Get drive input and apply to drivetrain
        keejLib::ChassVelocities velocities = controller.drive(1, keejLib::Controller::arcade);
        driveTrain.spinVolts(velocities);
        
        pros::delay(20);
    }
}
```

## Drive Modes

The Controller supports multiple drive control schemes:

<CodeGroup>

```cpp Arcade Drive
// Left stick Y for forward/back, right stick X for turning
auto velocities = controller.drive(1, keejLib::Controller::arcade);
driveTrain.spinVolts(velocities);
```

```cpp Tank Drive
// Left stick controls left side, right stick controls right side
auto velocities = controller.drive(1, keejLib::Controller::tank);
driveTrain.spinVolts(velocities);
```

```cpp Reverse Arcade
// Same as arcade but with reversed controls
auto velocities = controller.drive(1, keejLib::Controller::reverseArcade);
driveTrain.spinVolts(velocities);
```

```cpp Curvature Drive
// Curved response for smoother control
auto velocities = controller.drive(1, keejLib::Controller::curvature);
driveTrain.spinVolts(velocities);
```

</CodeGroup>

## Complete Example

Here's a full driver control example:

```cpp
#include "main.h"
#include "keejLib/lib.h"

// Hardware setup
pros::Controller prosController(pros::E_CONTROLLER_MASTER);
pros::MotorGroup leftMotors({-1, -2, -3});
pros::MotorGroup rightMotors({4, 5, 6});
pros::MotorGroup intake({-11, -12});

// keejLib components
keejLib::Controller controller(prosController);
keejLib::DriveTrainConstants dtConsts = {0.6, 600, 3.25};
keejLib::DriveTrain driveTrain(&leftMotors, &rightMotors, dtConsts);

void opcontrol() {
    while (true) {
        // Arcade drive control
        keejLib::ChassVelocities driveVelocities = 
            controller.drive(1, keejLib::Controller::arcade);
        driveTrain.spinVolts(driveVelocities);
        
        // Intake control
        if (prosController.get_digital(pros::E_CONTROLLER_DIGITAL_R1)) {
            intake.move_voltage(12000);
        }
        else if (prosController.get_digital(pros::E_CONTROLLER_DIGITAL_R2)) {
            intake.move_voltage(-12000);
        }
        else {
            intake.move_voltage(0);
        }
        
        pros::delay(20);
    }
}
```

## Direct Motor Control

You can also control the motor groups directly without DriveTrain:

```cpp
// Manual differential control
leftMotors.move_voltage(10000);   // Left side forward
rightMotors.move_voltage(-10000); // Right side backward (turn)

// Both sides forward
leftMotors.move_voltage(12000);
rightMotors.move_voltage(12000);
```
