  <h2 className="text-2xl sm:text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get keejLib integrated into your PROS project in minutes
  </p>
  
  <Steps>
    <Step title="Clone the repository">
      Add keejLib to your PROS project by cloning the repository into your project directory.
      
      ```bash
      cd your-pros-project
      git clone https://github.com/8pxl/keejLib.git lib
      ```
    </Step>
    
    <Step title="Include the library">
      Include keejLib in your source files. All functionality is under the `keejLib` namespace.
      
      ```cpp
      #include "keejLib/lib.h"
      
      // Access motor groups
      keejLib::DriveTrain dt(&leftMotors, &rightMotors, dtConsts);
      
      // Access chassis control
      keejLib::Chassis chassis(&dt, chassConsts, &imu, &vertEnc, &horizEnc);
      ```
    </Step>
    
    <Step title="Initialize your hardware">
      Set up your motors, sensors, and chassis with the appropriate constants for your robot.
      
      ```cpp
      // Define motor groups
      pros::MotorGroup leftMotors({-1, -2, -3});
      pros::MotorGroup rightMotors({4, 5, 6});
      
      // Create drivetrain
      keejLib::DriveTrainConstants dtConsts = {
        .gearRatio = 0.6,
        .cartridge = 600,
        .wheelDia = 3.25
      };
      
      keejLib::DriveTrain dt(&leftMotors, &rightMotors, dtConsts);
      ```
      
      <Note>
        You'll need to configure chassis constants specific to your robot's dimensions and tracking wheels. See the [setup guide](/chassis/setup) for details.
      </Note>
    </Step>
    
    <Step title="Start using advanced features">
      Use keejLib's powerful autonomous movement algorithms to control your robot.
      
      ```cpp
      // Move to a specific pose with heading control
      chassis.mtpose(
        {{48, 24}, keejLib::Angle(90, keejLib::HEADING)},
        12,
        params
      );
      
      // PID turn to angle
      chassis.turn(90, params);
      
      // Move to point with boomerang controller
      chassis.mtpoint({36, 36}, params);
      ```
    </Step>
  </Steps>
</div>
